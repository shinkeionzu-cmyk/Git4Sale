## 💳 Cara Pembelian & Akses
 Marketplace ini menggunakan model **Manual Fulfillment** untuk memastikan keselamatan kod sumber:

1.  **Pilih Sistem:** Pilih sistem yang anda perlukan di laman web marketplace.
2.  **Hubungi Developer:** Klik butang "Beli" untuk dihantar terus ke WhatsApp saya.
3.  **Pembayaran:** Lakukan pembayaran melalui pemindahan bank/e-wallet.
4.  **Akses GitHub:** Setelah pembayaran disahkan, sila berikan *Username GitHub* anda.
5.  **Collaborator:** Saya akan menambah anda sebagai *Collaborator* ke dalam repository private sistem tersebut untuk akses seumur hidup.
